# Springboard
This is my Rails 7 (Currently 7.2) starting point. Out of the box setup includes: 

- Ruby 3.2.0
- Devise Authentication (with Devise Invitable)
- Tailwind CSS
- Postgresql
On M1/M2 mac: 
    ```
    brew install libpq
    export PATH="/opt/homebrew/opt/libpq/bin:$PATH"
    gem install pg
    ```
